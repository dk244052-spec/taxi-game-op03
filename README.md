# taxi-game-op03

Проект по дисциплине ОП 03 Основы алгоритмизации и программирование

## Название:

"Игра Таксист"

## Проблема, которую решает мой проект

В современном мире существует много сложных игр, но простые текстовые игры на логику и ориентирование в пространстве помогают развивать внимательность и быстроту принятия решений. Игра "Таксист" предлагает пользователю простой и понятный геймплей, не требующий мощных ресурсов компьютера.

## Целевая аудитория:

школьники, студенты и взрослые люди, интересующиеся логическими играми

> 12-40 лет

## Цели и задачи моего проекта:

- Оформить введение проекта
- Сделать Use Case диаграмму и блок-схему
- Реализовать код игры с графическим интерфейсом
- Добавить систему логирования действий игрока
- Подготовиться к защите проекта
- Составить и оформить текстовую документацию в соответствии с ГОСТ
- Составить и оформить визуальную презентацию проекта в соответствии с ГОСТ

## Функционал:

- Управление такси по городу 5х5 с помощью клавиш w/a/s/d
- Подбор пассажира на карте
- Доставка пассажира к месту назначения
- Система заработка денег за успешные поездки
- Генерация новых пассажиров после каждой доставки
- Логирование всех действий в отдельный файл
- Выход из игры по клавише q
